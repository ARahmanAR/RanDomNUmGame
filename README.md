## How to Play

1. Open `index.html` in your web browser
2. Enter a number between 1 and 100 in the input field
3. Click "Submit Guess" or press Enter
4. Follow the hints (too high/too low) to narrow down your guess
5. Try to beat your best score!

## Running the Game

Simply open the `index.html` file in any modern web browser. No installation or build process required!
